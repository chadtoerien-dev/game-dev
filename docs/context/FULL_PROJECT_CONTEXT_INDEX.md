## Raw and packaged sources

The repository contains:

- `artifacts/packages/TheVeil_FullProjectArchive_0_6.zip` — verified bundle containing the exact uploaded `Pasted text.txt`, all historical generated ZIP deliverables from 0.1 through 0.5, and the audited Markdown context archive.
- The authoritative master prompt is also preserved verbatim as ordered Markdown parts under `docs/context/` for direct review and version control.

The nested historical ZIP files are delivery snapshots. Current source in the repository remains the development authority.

## Preservation approach

User-authored master prompts are preserved verbatim. Major assistant outputs are preserved canonically across dedicated documents and original package archives. The message ledger identifies whether each turn is verbatim, canonically reconstructed, summarised from prior-thread context or represented by a versioned package.

## Deliberate exclusions

The public archive excludes hidden reasoning, platform instructions, tool-internal payloads, unrelated personal data, account details, copyrighted reference-game assets and unlicensed music. These are not project deliverables and must not be committed to a public repository.
