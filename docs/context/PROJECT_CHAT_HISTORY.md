# The Veil — Project Chat History

This is the durable, project-focused history of the visible conversation that led to the current repository. It preserves user directions and records assistant outputs without including hidden reasoning, system instructions or unrelated personal information.

## 15 July 2026 — Browser prototype critique and Unreal decision

### User direction

The current browser visual was described as not looking good enough. The desired direction was:

- AAA-quality graphics and realism
- Faster, more dynamic and fluid gameplay
- A social-deduction structure with a group of faithful-style contestants and hidden conspirators
- If the group fails to banish a conspirator, the conspirators act at night
- An action-adventure RPG centred on interaction, voting, charisma, survival and deception

The user asked whether Unreal Engine 5 was free and whether a free or open-source equivalent existed, then selected Unreal Engine 5.

The user asked about platform support for console, PC, Mac, browser, mobile and VR/metaverse. The project direction became:

- Production game in Unreal Engine 5
- Windows PC first
- Browser access considered through Pixel Streaming rather than treating browser-native WebGL as the primary production build
- Architecture should not unnecessarily prevent later console support

The user then requested a master prompt for continuing the Unreal build in a new chat and added Hitman and the dramatic score of *Hitman 2: Silent Assassin* as references.

## 16 July 2026 — Master project brief supplied

The user supplied the complete master prompt now preserved in [`MASTER_PROMPT.md`](MASTER_PROMPT.md).

The prompt established:

- Working title: **The Veil**

- Two hidden conspirators and six loyal contestants

- Directed NPC-to-NPC relationships
- Information provenance

- Responsive third-person movement
- Tactical planning camera

- A phased roadmap and exact first implementation task
