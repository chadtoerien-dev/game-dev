# Project Context Archive

This directory preserves the design and development context that originally existed only in the ChatGPT project conversation.

## Authority order

When documents conflict, use this order:

1. [`MASTER_PROMPT.md`](MASTER_PROMPT.md) — authoritative original project brief.
2. [`REFERENCE_MUSIC_ADDENDUM.md`](REFERENCE_MUSIC_ADDENDUM.md) — authoritative gameplay-reference and score addendum.
3. [`DECISION_LOG.md`](DECISION_LOG.md) — accepted project decisions made after the prompts.
4. Current code, tests and integration documentation.
5. [`PROJECT_CHAT_HISTORY.md`](PROJECT_CHAT_HISTORY.md) — historical context; it explains how decisions emerged but is not itself a specification.

## Privacy boundary

Only game-project material from the visible conversation is preserved. System messages, hidden reasoning, unrelated personal context and account data are deliberately excluded. The repository is public, so future additions must follow the same rule.
